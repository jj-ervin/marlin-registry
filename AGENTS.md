# AGENTS.md — Cross-agent bootstrap

Read [`governance-commons/passes/dev-accord/DEV-ACCORD.03.md`](governance-commons/passes/dev-accord/DEV-ACCORD.03.md) before taking any action. That file is the canonical agent onboarding protocol for this portfolio — reading order, portfolio rules, ecosystem overview, and planning doc roles.

This file is the GC cross-agent bootstrap. Some runtimes auto-discover it; others need a host, adapter, or explicit instruction to read it. Any agent claiming GC conformance must read it before acting. All governance lives in `governance-commons/`.

For review and audit commands (normalize, validate, adversarial audit, trace, de-stale, close the loop), see [`docs/REVIEW-GRAMMAR.md`](docs/REVIEW-GRAMMAR.md).
