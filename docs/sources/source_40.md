# eco OG v0.2 — Inclusion Set (Merged: HIMSS26 + Global Standards)

## 0. Purpose
This file defines the **canonical inclusion primitives** for eco OG v0.2, derived from:
- HIMSS26 cross‑article synthesis  
- ISO/IEC 42001  
- NIST AI RMF  
- ISO/IEC 23894  
- ISO/IEC 27001  
- ISO/IEC 25010  
- HL7 FHIR  
- TEFCA  
- SAE autonomy levels (adapted for agents)  
- MITRE ATT&CK + DEFEND  
- COBIT 2019  

These primitives represent the **governance, safety, interoperability, and operational foundations** required for eco’s next evolution.
