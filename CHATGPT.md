# CHATGPT.md — ChatGPT session bootstrap

## Setup (one time)

Create a ChatGPT Project for this portfolio and attach these files:

1. `governance-commons/passes/dev-accord/DEV-ACCORD.03.md` — agent onboarding: reading order, portfolio rules, ecosystem overview
2. `governance-commons/passes/dev-accord/DEV-ACCORD.02.md` — execution model: PASS, RUN, Evidence Bundle

For multi-agent sessions, also attach:

- `governance-commons/agent-matrix/agent-matrix.yaml`

Once attached, every session in the project loads them automatically. No pasting required.

## When governance files change

Re-upload the changed files to the project. ChatGPT will use the updated versions.

This file is ChatGPT-specific. All portfolio governance lives in `governance-commons/`.
